API Reference

Base URL: https://api.tbook.com/v1

Authentication​

All /v1/* endpoints require an API key via Authorization header or x-api-key header.

Authorization: Bearer tbk_pub_YOUR_KEY
Key TypePrefixRate LimitAccess
Publishabletbk_pub_100 req/minRead + TX building
Secrettbk_secret_1000 req/minFull access + webhooks
Testtbk_test_UnlimitedAll features (devnet only)
Anonymous—30 req/minRead-only

Rate Limiting​

Rate limit information is included in response headers:

  • X-RateLimit-Limit — Maximum requests per window
  • X-RateLimit-Remaining — Remaining requests in current window
  • Retry-After — Seconds to wait (included on 429 responses)

Error Responses​

All errors follow a consistent format:

{
  "error": "Human-readable error message",
  "code": "VAULT_PAUSED"
}

HTTP Status Codes​

HTTP CodeMeaning
400Invalid request (missing fields, bad pubkey)
401Missing or invalid API key
422SDK validation error (vault paused, min deposit, etc.)
429Rate limit exceeded
500Internal server error

Error Codes​

CodeDescription
