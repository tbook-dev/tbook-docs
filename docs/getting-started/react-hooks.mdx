# React Hooks

The React Hooks package provides a declarative interface for integrating TBook Vault into React applications.

## Install

```bash
npm install @tbook-dev/vault-react @tbook-dev/vault-sdk \
  @solana/wallet-adapter-react @tanstack/react-query
```

## Setup Provider

```tsx
import { TBookVaultProvider } from "@tbook-dev/vault-react";
import { ConnectionProvider, WalletProvider } from "@solana/wallet-adapter-react";
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";

const queryClient = new QueryClient();

function App() {
  return (
    <QueryClientProvider client={queryClient}>
      <ConnectionProvider endpoint="https://api.devnet.solana.com">
        <WalletProvider wallets={[]} autoConnect>
          <TBookVaultProvider network="devnet">
            <YourApp />
          </TBookVaultProvider>
        </WalletProvider>
      </ConnectionProvider>
    </QueryClientProvider>
  );
}
```

## Use Hooks

```tsx
import {
  useVaultInfo,
  useSharePrice,
  useUserAccount,
  useDeposit,
  useRedeem,
  useClaim,
} from "@tbook-dev/vault-react";

function DepositForm() {
  const { data: vault } = useVaultInfo();
  const { data: price } = useSharePrice();
  const { data: user } = useUserAccount();
  const deposit = useDeposit();

  const handleDeposit = async () => {
    const sig = await deposit.mutateAsync(100); // 100 USDC
    console.log("Deposited!", sig);
  };

  return (
    <div>
      <p>APY: {price?.apy}%</p>
      <p>Your shares: {user?.effectiveShares}</p>
      <button onClick={handleDeposit} disabled={deposit.isPending}>
        {deposit.isPending ? "Depositing..." : "Deposit 100 USDC"}
      </button>
    </div>
  );
}
```

## Available Hooks

| Hook | Returns | Description |
|------|---------|-------------|
| `useVaultInfo()` | Vault state | TVL, epoch info, pause status |
| `useSharePrice()` | Price data | Current share price and APY |
| `useUserAccount()` | User balances | Shares, pending deposits/redeems, claimable USDC |
| `useDeposit()` | Mutation | Build, sign, and submit a deposit transaction |
| `useRedeem()` | Mutation | Build, sign, and submit a redeem transaction |
| `useClaim()` | Mutation | Build, sign, and submit a claim transaction |

All query hooks leverage `@tanstack/react-query` for caching, background refetching, and stale-while-revalidate behavior.
